JungleChat is a full-stack real-time one-to-one chat application built using the MERN stack (MongoDB, Express.js, React, Node.js) with Socket.IO for real-time communication.

The project was built as part of my internship-focused portfolio to demonstrate full-stack development skills, authentication systems, real-time communication, database design, API development, and modern frontend development.
